🎬 Dexter Video to GIF Converter Pro
제미나이(Gemini)와 함께 만든 초간편 영상-GIF 변환기!

mkv, mp4, mov, avi 등 다양한 영상 파일을 클릭 몇 번으로 움직이는 GIF(움짤)로 변환해 보세요.

✨ 주요 기능
다양한 포맷 지원: .mp4, .mkv, .mov, .avi 등 주요 영상 포맷 완벽 지원.

직관적인 UI: customtkinter를 활용한 세련되고 사용하기 쉬운 인터페이스.

드래그 앤 드롭: 파일을 직접 끌어다 놓아 빠르게 변환 가능 (tkinterdnd2 적용).

고화질 변환: MoviePy 엔진을 사용하여 화질 저하를 최소화한 GIF 생성.

🚀 설치 및 사용 방법
이 프로그램은 포터블(Portable) 버전으로 제작되었습니다. 복잡한 파이썬 환경 설정이나 라이브러리 설치 과정이 전혀 필요 없습니다!

[Releases] 페이지에서 최신 버전의 실행 파일(.exe)을 다운로드합니다.

별도의 설치 과정 없이, 다운로드한 파일을 그냥 실행합니다.

변환하고 싶은 영상을 선택하고 '변환' 버튼을 누르면 끝!
